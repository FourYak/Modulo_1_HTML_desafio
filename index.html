Estrutura Básica
Tabela com as tags para estrutura básica
Tag Definição Boas Práticas e SEO
<!DOCTYPE html> Declaração do tipo de documento (HTML5) Sempre deve ser a primeira linha do código.
<html> Elemento raiz do documento Inclui atributo lang ex: lang="pt-BR" para acessibilidade e SEO.
<head> Contêiner para metadados Deve conter <title>, <meta charset> e <meta name="viewport">
<body> Contêiner para conteúdo visível Dividir em seções emânticas (<header>, <main>, <footer>).

Metadados
Tabela com as tags para metadados
Tag Definição Boas Práticas e SEO
<title> Título da página(aparece nos resultados de busca) SEO Key: 50-60 caracteres incluir palavra-chave principal.
<meta charset="UTF-8"> Define a codificação de caracteres Obrigatório para evitar problemas de renderização.
<meta name="viewport"> CControla o layout em dispositivos móveis Usar content="width=device-width, initial-scale=1.0"
<meta name="description"> Descrição da página (snippet nos resultados de busca) SEO Key: 150-160 caracteres, chamativo e com palavras-chave.
<meta name="robots"> Instruções para rastreadores Ex: content="index, follow" (padrão) ou noindex para páginas privadas.
<link rel="canonical"> URL canônica para evitar conteúdo duplicado Fundamental para SEO em sites com URLs similares.
<link rel="icon"> Favicon da página Melhora a experiência do usuário.

Semântica Textual
Tabela com as tags para estrutura básica
Tag Definição Boas Práticas e SEO
<h1> a <h6> Hierarquia de títulos SEO Key: Usar apenas um <h1> por página, com palavras-chave. Subtópicos em <h2> ao <h6>.
<p> Parágrafo Textos longos melhoram SEO (mínimo 300 palavras).
<strong> Texto importante (semântico) Preferir sobre <b> para SEO.
<em> Ênfase (semântico) Preferir sobre <i> para acessibilidade.
<blockquote> Citação longa Usar para destacar citações relevantes.
<cite> Referência à fonte Melhora a credibilidade do conteúdo.

Tag Definição Boas Práticas e SEO
<a href="..."> Hiperlink SEO Key: Usar textos âncora descritivos (evitar "clique aqui").

Performance & SEO Técnico
Tabela com as tags para performance e seo técnico
Tag Definição Boas Práticas e SEO
<script defer> Carrega JS sem bloquear renderização Melhora velocidade de carregamento (Core Web Vitals).
<link rel="preload"> Pré-carrega recursos críticos Otimiza LCP (Largest Contentful Paint).
<noscript> Fallback para JS desativado Melhora acessibilidade e rastreamento.

Boas Práticas Gerais para SEO

  1. Hierarquia Semântica:

    • <h1> para o título principal

    • <h2> ao <h6> para subtópicos (não pular níveis)

  2. Imagens Otimizadas:

    • Sempre preencher <alt> com descrição relevante.

    • Comprimir arquivos (WebP para melhor perfomance).

  3. Links Internos:

    • Usar âncoras descritivas (ex: "veja nosso guida de HTML" em vez de "clique aqui").

  4. Conteúdo Prioritário

    • Colocar informações críticas no <main>

    • Evitar conteúdo importante em div sem semântica

    5. Mobile-First:

    • Usar <meta viewport> e em imagens responsívas (picture)

  5. Dados Estruturados:

    • Implementar Schema.org para rich spnippets (ex: avaliações, FAQs)

  6. Performance:

    • Adiar o carregamento de JS não crítico (<defer>/<async>)

    • Pré-carregar fontes e CSS crítico.
